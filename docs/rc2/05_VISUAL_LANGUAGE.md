# RC2 — Visual Language

**Status:** DRAFT — pending Paymark reference audit

---

## Brand Principles

1. **Enterprise confidence** — BOSS looks like it belongs in the Fortune 500, used by Main Street
2. **Earned simplicity** — complexity is hidden, not removed
3. **Kinetic precision** — motion communicates meaning, never decoration
4. **Dark authority** — dark theme is the default; it signals power

## Visual Hierarchy

Headlines → Subheadlines → Body → Labels → Metadata

Every screen must have one and only one dominant element.

## Iconography

- Style: Line icons, 1.5px stroke, rounded caps
- Size: 16px (ui), 20px (feature), 24px (section), 48px (hero)
- Source: TBD after audit (Lucide / Heroicons / custom)

## Illustration

- Style: TBD after Paymark audit
- Tone: Abstract + technical (not playful, not corporate stock)

## Photography / Video

- No stock photography
- Product screenshots / screen recordings only
- Dark-mode UI recordings preferred

## Logo Usage

- Full lockup: wordmark + icon
- Compact: icon only (favicons, app icon)
- Reversed: white on dark (primary usage)
- Colors: never render logo in accent red — only white or neutral
